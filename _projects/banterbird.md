## Overview

Banterbird is a web application built as part of the ASCTE 301/401 curriculum. Inspired by Twitter, it allows users to post short messages and interact with a feed. The backend is written in Python and integrates with external APIs.

## Features

- User post creation and feed display
- Python-powered backend
- API integration for extended functionality
- Web-based front end

## Tools & Skills Used

- **Python** — backend logic and API handling
- **Web Development** — front end structure and design
- **APIs** — integrating external services into the app

## What I Learned

This was my first full-stack web project. Building both the front end and the backend taught me how the two sides communicate, and working with APIs showed me how much functionality you can add to an app by connecting to existing services rather than building everything from scratch.
