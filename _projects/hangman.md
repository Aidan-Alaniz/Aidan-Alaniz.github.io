## Overview

A classic Hangman game playable in the terminal, written in Python. The player guesses letters to reveal a hidden word before running out of attempts.

## Features

- Random word selection
- Letter guess tracking — shows correct and incorrect guesses
- Visual attempt counter
- Win/loss detection

## Tools & Skills Used

- **Python** — game logic, string manipulation, user input

## What I Learned

A straightforward project that helped me get comfortable with string operations and managing program state. Good practice for thinking about how user input maps to game outcomes.
